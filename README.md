## 安装

可以使用包管理器（如npm或yarn）安装组件，也可以从存储库安装组件。

### npm
```bash
npm i basic-vue-chat
```

### yarn
```bash
yarn add basic-vue-chat
```

### using repository
```bash
git clone https://github.com/unclepan/uncle-vue-chat.git
cd uncle-vue-chat
npm i
```
